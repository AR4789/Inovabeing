# 🚀 Inovabeing – Cloud-Native Full-Stack Application

A cloud-native application designed and deployed using **Google Cloud Platform (GCP)** with **Infrastructure as Code (Terraform)** and automated **CI/CD pipelines (GitHub Actions)**.

---

## 🌟 Highlights

- ☁️ Deployed on **GCP** with scalable cloud architecture  
- 🏗️ Infrastructure managed using **Terraform (IaC)**  
- ⚙️ Automated deployments via **GitHub Actions CI/CD**  
- 🔗 Backend APIs built using **Node.js**  
- 🌐 Full-stack application with production-like setup  

---

## 🏗️ Architecture Overview

User → Frontend → Backend (Node.js APIs) → GCP Services
↓
Terraform-managed Infra
↓
CI/CD via GitHub Actions


---

## ⚙️ Tech Stack

- **Backend:** Node.js
- **Frontend:** React
- **Cloud:** Google Cloud Platform (GCP)  
- **Infrastructure:** Terraform  
- **CI/CD:** GitHub Actions  
- **Version Control:** Git & GitHub  

---

## ☁️ Infrastructure (Terraform)

This project uses Terraform to provision and manage:

- Compute resources (VM / services)
- Networking configuration
- Deployment setup

> Infrastructure is version-controlled and reproducible.

---

## 🔄 CI/CD Pipeline

Implemented using **GitHub Actions**:

- Automated build & deployment workflows  
- Triggered on code push  
- Reduces manual deployment effort
